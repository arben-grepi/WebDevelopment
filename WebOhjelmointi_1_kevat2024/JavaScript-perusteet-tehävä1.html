Palindromi-tehtävä

Suorita tehtävä selaimen konsolissa

Kopioi alla oleva JavaScript-koodi ja liitä se selaimen konsoliin. Paina sitten Enter suorittaaksesi tehtävän.

Funktio palauttaa true/false sen perusteella onko sana palindromi. 

function palindromi(merkkijono) {
  if (merkkijono.length == 0) return false;
  var merkkijono2 = "";

  let i = merkkijono.length - 1;
  while (i >= 0) {
    merkkijono2 += merkkijono[i];
    i--;
  }
  if (merkkijono == merkkijono2) return true;
  else return false;
}

var merkkijono = prompt("Anna merkkijono: ");
console.log(palindromi(merkkijono));
